# Icons

Place your extension icons here:
- `icon16.png` (16x16)
- `icon48.png` (48x48)
- `icon128.png` (128x128)

## Quick Setup

1. Open `scripts/generate-icons.html` in your browser
2. Right-click each canvas and save as PNG with the correct name
3. Place the PNG files in this `icons/` folder

## Alternative

You can create your own icons using any image editor. The icons should:
- Be square (16x16, 48x48, 128x128)
- Use PNG format
- Have a transparent or solid background
- Represent a bookmark or decision-related concept

**Note**: The extension will work without icons, but Chrome will show a default icon and may display warnings.
